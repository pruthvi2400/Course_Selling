# Course Selling Platform (Full Stack)

A full-stack course selling application built using **Angular**, **Node.js (Express)**, and **MongoDB**.  

Users can browse courses, view details, and purchase courses. Admins can manage courses through protected routes.

---

## Tech Stack

### Frontend
- Angular
- TypeScript
- RxJS
- Angular Router
- SCSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- CORS

### Database
- MongoDB (Local / Compass)

---

## Features

### User
- Browse courses
- View course details
- Purchase courses
- Authentication (Login / Signup)

### Admin
- Create courses
- Edit courses
- Protected admin routes

### System
- REST API architecture
- Angular routing
- Role-based access control
- MongoDB integration
